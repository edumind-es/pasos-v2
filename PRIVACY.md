# Privacy

This public source release is intended to contain code, documentation,
tests and synthetic examples only.

It must not contain production databases, backups, uploaded classroom
files, real user records, access logs, analytics exports or private
configuration.

Operators of deployed instances are responsible for their own privacy
and data-protection obligations, including GDPR/RGPD compliance, lawful
basis, retention periods, user information, data-subject rights, processor
agreements and secure deletion of backups.

For local development, use synthetic data and placeholder credentials.
